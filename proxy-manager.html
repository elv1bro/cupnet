▸ Template syntax guide
{RAND:min-max} Random integer in range, generated fresh on every Connect.
Example: {RAND:40001-49999} → 43217
{SID} Ephemeral session ID — shown in the Variables table, never stored in the profile.
Leave blank → auto-generates cupnet + 10 random digits.
Change it at any time without saving a new profile.
Example: {SID} → cupnet3849201745
{VARNAME} Saved variable — stored with the profile (e.g. country code, username suffix).
Name must start with a letter, can contain letters, digits, underscore.
Example: {COUNTRY} → ru
Full example (rotating residential proxy)
http://user-{SID}:p@ssw0rd@{COUNTRY}.proxy-provider.com:{RAND:10000-19999}
