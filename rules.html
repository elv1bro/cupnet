Request Interceptor

Intercept rules modify or block requests before they reach the server. By default use a glob with *, e.g. *example.com/api*. If CUPNET_INTERCEPT_STRICT_URLS=1 is set, use only a URL prefix without *, e.g. https://api.example.com/v1/.
Request will be cancelled before it is sent. No additional parameters needed.
