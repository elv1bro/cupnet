Request Interceptor

Intercept rules modify or block requests before they reach the server. По умолчанию: glob с *, напр. *example.com/api*. Если задано CUPNET_INTERCEPT_STRICT_URLS=1 — только префикс URL без *, напр. https://api.example.com/v1/.
Request will be cancelled before it is sent. No additional parameters needed.
